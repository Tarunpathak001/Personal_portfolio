Projects Worked Upon

Mini Docker

Mini-Docker is an educational container runtime built from scratch in Python to understand how modern containers work at the Linux kernel level. The project implements core container primitives such as Linux namespaces, cgroups v2, and Seccomp-BPF to provide process isolation, resource control, and security. It supports container lifecycle management through a custom CLI, along with features like rootless containers, virtual networking, and OCI-compatible image handling.

View Code on GitHub

Mini-Redis-Cassandra

Mini-Redis-Cassandra is an educational distributed, in-memory key-value database built from scratch in Python to understand core distributed systems concepts. The project implements clustering via gossip-based membership, sharding using consistent hashing, and tunable consistency levels inspired by Redis and Cassandra. It also includes persistence through write-ahead logging and snapshots, along with basic fault tolerance and read repair mechanisms.

View Code on GitHub

Code Cache

CodeCache is a local network package caching system designed to reduce dependency download time and bandwidth usage for development teams. It caches npm and Python packages on a shared server, allowing packages to be downloaded once and reused across machines. The project includes a web-based dashboard for monitoring cache usage and supports automated cleanup and Windows service deployment.

View Code on GitHub

Bookmark Manager

Bookmark Manager is a browser extension designed to make bookmark organization fast and intuitive. It supports folder-based organization, tagging, search, and drag-and-drop reordering within a clean, responsive interface. The project focuses on improving usability and efficiency for managing large collections of bookmarks directly inside the browser.
